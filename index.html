📺 OBS Integration

Add the live overlay to OBS Studio so your stream shows real-time court fixtures and scores. The overlay reads the exported data file and refreshes every 2 seconds.

Setup Steps
1
Download the obs_overlay.html file that came with this app and save it somewhere on your computer.
2
In OBS, add a new Browser Source. Check Local file and select your obs_overlay.html.
3
Set the browser source dimensions to 1920 × 1080 for full HD, or match your stream canvas size.
4
Click Export Data (top right) after every round — the overlay will automatically reload and show the latest data.
5
Alternatively use the leaderboard.html file as a Browser Source for a live standings view that auto-refreshes.
Export & Save

Export the current tournament state to a ctc_data.json file. Place this file in the same folder as your overlay HTML files so they can read it.

Live Leaderboard

Open leaderboard.html in a browser window or as an OBS Browser Source for a live scrolling standings board.

⚠️Both the leaderboard and overlay look for ctc_data.json in the same folder. Re-export after each round update.
⚡ Tiebreak Required

Both teams have the same total score and equal top individual scores. Ask an external question and award an advantage point to the winning team. This does not count towards totals — it only determines advancement.
