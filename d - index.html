📺 OBS Integration
Setup Steps
1
Keep obs_overlay.html and leaderboard.html in the same folder as this app.
2
In OBS add a Browser Source → Local file → select obs_overlay.html. Set to 1920×1080.
3
Click ⬇ Export after each round — the overlay auto-refreshes from ctc_data.json every 2 seconds.
4
Use leaderboard.html as a second Browser Source for live standings.
